## vue2-lagou

[![996.icu](https://img.shields.io/badge/link-996.icu-red.svg)](https://996.icu)

> vue2 仿拉勾网的单页应用
> 数据来自拉勾网


#### 技术栈 vue2 + vue-router + webpack + less

### Build Setup

``` bash
# install dependencies
npm install

# 开发环境
# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
